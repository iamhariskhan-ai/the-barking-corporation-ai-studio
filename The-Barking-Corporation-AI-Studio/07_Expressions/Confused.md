# Confused Expression

## Description
A facial expression showing uncertainty, lack of understanding, doubt, or curiosity about something unclear.

## Emotion
Confused, uncertain, puzzled, questioning, unsure

## Facial Features
- Slightly raised eyebrows
- Furrowed forehead
- Slightly tilted head
- Partially open mouth
- Focused but uncertain eyes
- Questioning facial posture

## Usage
- Understanding problems
- Asking questions
- Unexpected situations
- Learning new concepts
- Reactions to unclear information
