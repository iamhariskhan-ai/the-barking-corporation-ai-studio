# Thinking Expression

## Description
A thoughtful facial expression showing concentration, analysis, curiosity, or deep consideration.

## Emotion
Thoughtful, curious, focused, analytical, reflective

## Facial Features
- Slightly narrowed eyes
- Raised or relaxed eyebrows
- Hand near chin (optional)
- Neutral or slightly serious mouth
- Focused gaze
- Calm facial posture

## Usage
- Problem solving
- Decision making
- Learning moments
- Creative brainstorming
- Planning and analysis
