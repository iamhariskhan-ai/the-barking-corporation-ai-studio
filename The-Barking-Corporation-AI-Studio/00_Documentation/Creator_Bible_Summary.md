# Vision

The Barking Corporation is a recurring AI-generated animated comedy series that follows a group of talking animal coworkers navigating the everyday challenges of working in a modern software company.

The goal is to create highly engaging, family-friendly short-form videos that are entertaining, relatable, visually consistent, and scalable for long-term content production.

Every episode should feel like it belongs to the same animated universe.

---

# Mission

Build a professional AI-powered animation studio capable of producing high-quality, reusable, and consistent animated episodes using AI tools such as:

- ChatGPT
- Gemini Veo
- AI Voice Generation
- Video Editing Software
- YouTube Studio

The production system should prioritize:

- Quality
- Consistency
- Automation
- Scalability
- Reusability

# Studio Values

The studio is built around five principles:

## 1. Consistency

Characters, environments, voices, and animation should remain visually and behaviorally consistent.

---

## 2. Simplicity

One episode should focus on one joke, one situation, and one clear resolution.

---

## 3. Reusability

Characters, environments, prompts, and assets should be reusable across hundreds of episodes.

---

## 4. Quality

Every episode should meet a professional production standard before publishing.

---

## 5. Scalability

The production system should support creating hundreds or thousands of episodes with minimal manual effort.

# Storytelling Guidelines

Every episode should:

- Begin with an engaging hook.
- Introduce one central problem.
- Build toward a simple conflict.
- End with a memorable punchline.
- Deliver value in under 45 seconds.

Avoid complex storylines.

# AI Production System

The production pipeline uses modular prompts rather than large documents.

Each Gemini Veo scene combines:

- Master Studio Prompt
- Character Modules
- Environment Module
- Camera Module
- Lighting Module
- Audio Module
- Scene Prompt
- Negative Prompt

This keeps prompts concise, reusable, and consistent.
