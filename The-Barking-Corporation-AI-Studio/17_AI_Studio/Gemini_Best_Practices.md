# Purpose

Follow these guidelines to generate the highest-quality Gemini Veo videos.

---

# Best Practices

- Use one scene per prompt
- Keep clips between 8–15 seconds
- Use 1–3 main characters
- One primary action per scene
- One primary emotion per character
- Use consistent environments
- Use cinematic camera shots
- Keep dialogue short and natural
- Maintain continuity across scenes
- Always reference project modules

---

# Avoid

- Too many characters
- Multiple locations in one scene
- Rapid camera changes
- Long monologues
- Conflicting emotions
- Overloaded prompts
