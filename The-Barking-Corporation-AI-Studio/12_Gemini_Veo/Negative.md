# Negative Prompt

Avoid

- Character inconsistency
- Outfit changes
- Wrong environment
- Wrong lighting
- Camera shake
- Motion blur
- Robotic animation
- Floating objects
- Deformed faces
- Extra limbs
- Low-quality textures
- Incorrect proportions
- Poor lip sync
- Unrealistic expressions
- Background artifacts
- Text or watermarks

---

Goal

Generate clean, high-quality Pixar-inspired cinematic output.
