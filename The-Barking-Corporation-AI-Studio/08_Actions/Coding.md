# Purpose

Defines the standard coding animation used by software engineers while developing features, fixing bugs, reviewing pull requests, debugging applications, or writing automation scripts.

Coding is the core action of The Barking Corporation and should convey concentration, problem-solving, and professionalism.

---

# Description

Characters actively work at their computer, alternating between typing, reading code, scrolling, using shortcuts, and occasionally pausing to think.

Coding should feel natural and uninterrupted, with realistic workstation interactions.

---

# Body Movement

- Upright seated posture
- Slight forward lean
- Relaxed shoulders
- Stable torso
- Minimal unnecessary movement
- Comfortable ergonomic position

Avoid excessive leaning or exaggerated movements.

---

# Head Movement

- Eyes focused on monitor
- Small head tilts while reading code
- Occasional nod after understanding something
- Brief glance toward a second monitor (if available)

Head movement should remain subtle and purposeful.

---

# Hand Movement

- Both hands on keyboard during coding
- Frequent keyboard shortcuts
- Smooth mouse movement
- Occasional scrolling
- Natural pauses while thinking
- One hand briefly moves to the mouse when navigating code

Hands should always align naturally with the keyboard and mouse.

---

# Eye Direction

Characters should focus on:

- Code editor
- Terminal
- Browser
- Debug console
- Pull request
- Documentation

Eyes should remain engaged with the current task.

# Gemini Prompt Module

Anthropomorphic software engineer coding at a modern dual-monitor workstation, smooth keyboard typing, natural mouse interaction, focused on a code editor and terminal, ergonomic posture, premium corporate office, Pixar-inspired 3D animation, expressive performance, cinematic quality, highly detailed.

# Negative Prompt

Avoid:

- Random typing
- Floating hands
- Keyboard clipping
- Excessive body movement
- Robotic animation
- Unrealistic coding speed
- Looking away constantly
- Poor posture
- Empty monitor
- Jerky mouse movement
