# Purpose

Defines the standard typing animation used by all characters while working on computers, writing emails, coding, chatting, documenting, or responding to customers.

Typing should appear smooth, realistic, and professional while reflecting each character's personality.

This is one of the most frequently used actions throughout the series.

---

# Description

Characters interact naturally with a keyboard while maintaining focus on the monitor.

Typing should feel purposeful and synchronized with the work being performed.

Movements should remain subtle without exaggerated hand or body motion.

---

# Body Movement

- Upright seated posture
- Relaxed shoulders
- Slight forward lean
- Stable torso
- Minimal unnecessary movement
- Comfortable sitting position

Avoid excessive body swaying.

---

# Head Movement

- Eyes directed toward monitor
- Small head adjustments
- Occasional nod while reading
- Natural neck movement

Avoid frequent head turning.

---

# Hand Movement

- Both hands positioned on keyboard
- Natural finger movement
- Smooth key presses
- Occasional movement to mouse
- Brief pauses while thinking

Hands should never appear to float above the keyboard.

---

# Eye Direction

Characters should focus on:

- Computer monitor
- Code editor
- Email
- Design software
- Chat window
- Documentation

Occasionally glance toward teammates during conversations.

# Negative Prompt

Avoid:

- Random key smashing
- Floating hands
- Keyboard clipping
- Robotic movement
- Excessive arm motion
- Unrealistic typing speed
- Looking away constantly
- Poor posture
- Jerky animation
- Unsynchronized finger movement
