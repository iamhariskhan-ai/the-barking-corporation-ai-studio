Version: 1.0

Project: The Barking Corporation

---

# Purpose

Defines the technical ambience for DevOps, infrastructure, deployment, and monitoring scenes.

---

# Mood

- Technical
- Focused
- Quiet
- Professional

---

# Background Ambience

- Continuous server fan hum
- Cooling systems
- Network equipment
- Soft electronic beeps
- Air circulation

---

# Recommended SFX

- Keyboard typing
- Server alert
- Terminal notification
- Rack door
- Network beep

---

# Volume

Background: 20%

Dialogue: 100%

Alerts: 40%

---

# Best Used For

- Deployment
- Monitoring
- Infrastructure
- Incident Response
- Server Maintenance

---

# Gemini Prompt

Professional server room ambience with continuous server fan hum, cooling systems, subtle electronic equipment sounds, modern data center atmosphere.

---

# Negative Prompt

Avoid:

- Factory noise
- Heavy machinery
- Generator sounds
- Alarm loops
- Loud fans
