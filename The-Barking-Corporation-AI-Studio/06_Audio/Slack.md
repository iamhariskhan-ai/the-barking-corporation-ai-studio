# Slack Notification

Version: 1.0

Project: The Barking Corporation

Module: Audio SFX

Status: Production Ready

---

# Purpose

Represents modern workplace communication through Slack notifications.

Used to indicate:

- New message
- Team mention
- Task assignment
- Build notification
- Sprint reminder
- Channel activity

---

# Mood

Professional

Modern

Instant

Subtle

---

# Sound Characteristics

- Short digital notification
- Soft chime
- Clean UI sound
- Non-intrusive
- Duration: 0.3–0.8 seconds

---

# Volume

30–40%

Should never overpower dialogue.

---

# Best Used For

- Buddy receives updates
- Sprint reminder
- Build completed
- New task assigned
- Team discussion

---

# Gemini Prompt

Soft modern workplace notification sound, subtle digital message alert, professional collaboration platform notification, clean UI audio.

---

# Negative Prompt

Avoid:

- Phone ringtone
- Gaming notification
- Alarm sound
- Cartoon effects
- Loud chimes
- Echo

---

# QA Checklist

☐ Short notification

☐ Clean sound

☐ Professional tone

☐ No distraction

☐ Matches on-screen action
